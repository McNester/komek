# Komek


## This is Komek- Mental health support chat bot
People can use this bot for getting mental support. This project uses Result Augumented Pipeline with llama 3.2 for better responses and more sentient/empathic support



## Running the project, the project uses docker containers for deployment and running. 


1. Install and run ollama docker container:
```bash
docker run -d -v ollama/ollama -p 11434:11434 ollama/ollama
```

2. Start the Komek bot
```
docker-compose up --build
```
